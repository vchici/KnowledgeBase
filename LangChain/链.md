# Chains（链）

将多个组件串联成工作流，实现复杂逻辑

## LCEL（LangChain Expression Language）

使用管道操作符 `|` 连接组件

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser

# 定义链
chain = prompt | model | parser

# 执行
result = chain.invoke({"question": "什么是Python"})
```

## 基本原理

每个组件都实现了统一的接口：
- `invoke` - 同步执行
- `ainvoke` - 异步执行
- `stream` - 流式执行
- `batch` - 批量执行

## 简单链

```python
from langchain_openai import ChatOpenAI
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser

model = ChatOpenAI()
prompt = ChatPromptTemplate.from_template("翻译成英文：{text}")
parser = StrOutputParser()

chain = prompt | model | parser

result = chain.invoke({"text": "你好"})
```

## RunnablePassthrough

传递数据，用于数据透传或赋值

```python
from langchain_core.runnables import RunnablePassthrough

chain = {
    "original": RunnablePassthrough(),  # 透传原始输入
    "processed": prompt | model | parser
}
```

## RunnableParallel

并行执行多个链

```python
from langchain_core.runnables import RunnableParallel

chain = RunnableParallel(
    summary=prompt1 | model | parser,
    keywords=prompt2 | model | parser
)

result = chain.invoke({"text": "长文章..."})
# 输出: {"summary": "...", "keywords": [...]}
```

## RunnableLambda

将自定义函数包装为Runnable

```python
from langchain_core.runnables import RunnableLambda

def process(text):
    return text.upper()

chain = prompt | model | parser | RunnableLambda(process)
```

## 链的组合

```python
# 串行组合
chain = chain1 | chain2 | chain3

# 嵌套组合
chain = {
    "step1": chain1,
    "step2": chain2
} | final_chain
```

## 完整示例

```python
from langchain_openai import ChatOpenAI
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser
from langchain_core.runnables import RunnablePassthrough, RunnableParallel

model = ChatOpenAI()

# 定义prompt
summary_prompt = ChatPromptTemplate.from_template("总结：{text}")
keywords_prompt = ChatPromptTemplate.from_template("提取关键词：{text}")

# 构建链
chain = RunnableParallel(
    text=RunnablePassthrough(),
    summary=summary_prompt | model | StrOutputParser(),
    keywords=keywords_prompt | model | StrOutputParser()
)

result = chain.invoke("这是一篇长文章...")
```

## 链的优势

1. **可组合** - 像搭积木一样组合
2. **统一接口** - invoke/stream/batch
3. **可调试** - 每个步骤可追踪
4. **可复用** - 组件独立，易于复用
