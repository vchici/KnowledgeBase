# Tools（工具）深入

自定义工具，扩展模型能力边界

## 回顾：Agent执行流程

```
用户提问 → 模型分析(Reasoning) → 调用工具 → 分析结果(Observation) → 输出
```

模型本身不能执行工具，返回的是工具调用意图，LangChain负责执行

## 定义工具

### 使用 @tool 装饰器

```python
from langchain_core.tools import tool

@tool
def get_weather(city: str) -> str:
    """获取指定城市的天气信息
    
    Args:
        city: 城市名称
        
    Returns:
        天气信息
    """
    # 模拟API调用
    weather_data = {"北京": "晴天", "上海": "多云"}
    return weather_data.get(city, "未知城市")

# 工具信息
print(get_weather.name)      # get_weather
print(get_weather.description)  # 获取指定城市的天气信息...
print(get_weather.args_schema)  # 参数schema
```

### 使用 StructuredTool

```python
from langchain_core.tools import StructuredTool
from pydantic import BaseModel, Field

class CalculatorInput(BaseModel):
    a: int = Field(description="第一个数字")
    b: int = Field(description="第二个数字")

def multiply(a: int, b: int) -> int:
    return a * b

tool = StructuredTool.from_function(
    func=multiply,
    name="multiply",
    description="两数相乘",
    args_schema=CalculatorInput
)
```

### 使用 Pydantic 定义参数

```python
from langchain_core.tools import tool
from pydantic import BaseModel, Field

class SearchInput(BaseModel):
    query: str = Field(description="搜索关键词")
    limit: int = Field(default=5, description="返回数量")

@tool(args_schema=SearchInput)
def search(query: str, limit: int = 5) -> str:
    """搜索知识库"""
    return f"找到{limit}条关于{query}的结果"
```

## 将工具绑定到模型

```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI()
tools = [get_weather, search]

# 绑定工具
model_with_tools = model.bind_tools(tools)

# 调用
response = model_with_tools.invoke("北京天气怎么样？")

# 检查是否有工具调用
if response.tool_calls:
    print(response.tool_calls)
    # [{'name': 'get_weather', 'args': {'city': '北京'}, 'id': 'call_xxx'}]
```

## 工具执行

### 手动执行

```python
tool_call = response.tool_calls[0]
tool_name = tool_call["name"]
tool_args = tool_call["args"]

# 查找并执行
for tool in tools:
    if tool.name == tool_name:
        result = tool.invoke(tool_args)
        break
```

### 使用 ToolNode（LangGraph）

```python
from langgraph.prebuilt import ToolNode

tool_node = ToolNode(tools)
```

## 完整Agent示例

```python
from langchain_openai import ChatOpenAI
from langchain_core.tools import tool
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_core.messages import AIMessage, HumanMessage, ToolMessage

@tool
def get_weather(city: str) -> str:
    """获取城市天气"""
    weather_data = {"北京": "晴天 25°C", "上海": "多云 22°C"}
    return weather_data.get(city, "未知城市")

@tool
def calculate(expression: str) -> str:
    """计算数学表达式"""
    try:
        return str(eval(expression))
    except:
        return "计算错误"

# 创建模型和工具
model = ChatOpenAI()
tools = [get_weather, calculate]
model_with_tools = model.bind_tools(tools)

# 对话循环
messages = []
while True:
    user_input = input("你: ")
    if user_input == "quit":
        break
    
    messages.append(HumanMessage(user_input))
    
    # 循环处理工具调用
    while True:
        response = model_with_tools.invoke(messages)
        messages.append(response)
        
        if not response.tool_calls:
            print(f"AI: {response.content}")
            break
        
        # 执行工具
        for tool_call in response.tool_calls:
            for tool in tools:
                if tool.name == tool_call["name"]:
                    result = tool.invoke(tool_call["args"])
                    messages.append(ToolMessage(result, tool_call_id=tool_call["id"]))
                    break
```

## 工具类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 查询工具 | 获取信息 | 搜索、天气、数据库查询 |
| 操作工具 | 执行操作 | 发送邮件、调用API |
| 计算工具 | 数据处理 | 计算器、数据转换 |
| 系统工具 | 系统交互 | 文件操作、命令执行 |

## 工具设计原则

1. **单一职责** - 每个工具只做一件事
2. **清晰描述** - 让模型理解何时使用
3. **参数明确** - 类型注解和描述完整
4. **错误处理** - 返回有意义的错误信息
5. **幂等性** - 相同输入产生相同结果（如可能）
