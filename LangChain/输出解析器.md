# Output Parsers（输出解析器）

将模型输出解析为结构化数据

## 为什么需要

模型返回的是字符串，但业务通常需要：
- JSON对象
- 列表
- 特定格式数据

## 基本使用

```python
from langchain_core.output_parsers import StrOutputParser

# 最简单的解析器，直接返回字符串
parser = StrOutputParser()
result = parser.parse("模型输出")
```

## JsonOutputParser

解析JSON格式输出

```python
from langchain_core.output_parsers import JsonOutputParser
from langchain_core.prompts import PromptTemplate
from pydantic import BaseModel, Field

# 定义数据结构
class User(BaseModel):
    name: str = Field(description="用户名")
    age: int = Field(description="年龄")

# 创建解析器
parser = JsonOutputParser(pydantic_object=User)

# 在prompt中注入格式说明
prompt = PromptTemplate.from_template(
    "提取用户信息：{query}\n{format_instructions}"
).partial(format_instructions=parser.get_format_instructions())

# 使用
chain = prompt | model | parser
result = chain.invoke({"query": "张三今年25岁"})
# 输出: {"name": "张三", "age": 25}
```

## PydanticOutputParser

基于Pydantic模型验证和解析

```python
from langchain_core.output_parsers import PydanticOutputParser

parser = PydanticOutputParser(pydantic_object=User)
# 自动生成格式说明，自动验证类型
```

## CommaSeparatedListOutputParser

解析逗号分隔的列表

```python
from langchain_core.output_parsers import CommaSeparatedListOutputParser

parser = CommaSeparatedListOutputParser()
result = parser.parse("苹果, 香蕉, 橘子")
# 输出: ['苹果', '香蕉', '橘子']
```

## 与Prompt配合

解析器可以生成格式说明，注入到prompt中指导模型输出：

```python
parser = JsonOutputParser(pydantic_object=User)

prompt = PromptTemplate.from_template(
    "{query}\n{format_instructions}"
).partial(format_instructions=parser.get_format_instructions())
```

## 流式解析

支持流式输出的解析

```python
# 流式解析JSON
async for chunk in (prompt | model | parser).astream({}):
    print(chunk)
```
