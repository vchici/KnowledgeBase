# Memory（记忆管理）

管理对话历史，实现多轮对话

## 核心问题

模型API是无状态的，每次请求需要传入完整对话历史

## RunnableWithMessageHistory

LCEL方式管理对话历史

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_openai import ChatOpenAI
from langchain_core.output_parsers import StrOutputParser
from langchain_core.runnables import RunnableWithMessageHistory
from langchain_community.chat_message_histories import ChatMessageHistory

# 定义prompt，包含历史消息占位符
prompt = ChatPromptTemplate.from_messages([
    ("system", "你是一个{role}"),
    MessagesPlaceholder("chat_history"),
    ("human", "{question}")
])

model = ChatOpenAI()
parser = StrOutputParser()

chain = prompt | model | parser

# 存储历史
store = {}

def get_session_history(session_id):
    if session_id not in store:
        store[session_id] = ChatMessageHistory()
    return store[session_id]

# 包装链
chain_with_history = RunnableWithMessageHistory(
    chain,
    get_session_history,
    input_messages_key="question",
    history_messages_key="chat_history"
)

# 使用
chain_with_history.invoke(
    {"role": "助手", "question": "你好"},
    config={"configurable": {"session_id": "user_001"}}
)
```

## ChatMessageHistory

内存存储对话历史

```python
from langchain_community.chat_message_histories import ChatMessageHistory

history = ChatMessageHistory()
history.add_user_message("你好")
history.add_ai_message("你好！有什么可以帮你的？")

# 获取所有消息
messages = history.messages
```

## 持久化存储

### 文件存储

```python
from langchain_community.chat_message_histories import FileChatMessageHistory

history = FileChatMessageHistory("chat_history.json")
```

### Redis存储

```python
from langchain_community.chat_message_histories import RedisChatMessageHistory

history = RedisChatMessageHistory(
    session_id="user_001",
    url="redis://localhost:6379"
)
```

### 数据库存储

```python
from langchain_community.chat_message_histories import SQLChatMessageHistory

history = SQLChatMessageHistory(
    session_id="user_001",
    connection_string="sqlite:///chat.db"
)
```

```python
from langchain_core.messages import trim_messages

trimmed = trim_messages(
    history.messages,
    max_tokens=1000,
    strategy="last",  # 保留最后N条
    token_counter=len,
    include_system=True
)
```

## 总结窗口

只保留最近N轮对话

```python
from langchain_community.chat_message_histories import ChatMessageHistory

class WindowChatMessageHistory(ChatMessageHistory):
    def __init__(self, window_size=5):
        super().__init__()
        self.window_size = window_size
    
    def add_message(self, message):
        super().add_message(message)
        if len(self.messages) > self.window_size * 2:
            self.messages = self.messages[-self.window_size * 2:]
```

## 记忆类型对比

| 方式 | 特点 | 适用场景 |
|------|------|----------|
| 内存 | 快速、易失 | 测试、短期会话 |
| 文件 | 简单持久化 | 单机应用 |
| Redis | 高性能、分布式 | 生产环境 |
| 数据库 | 可靠、可扩展 | 企业应用 |
