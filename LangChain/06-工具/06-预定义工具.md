# 网络搜索工具tavily

- 消耗token过多
- 生成的结果没有引用来源

## 消耗token过多

**`max_results`** 设为 `2` 或 `3`

绝对不要开启 **`include_raw_content=True`**

检查 Agent 是否陷入了**死循环搜索**（同一个问题反反复复搜好几次）。如果是，需要优化 Agent 的 `System Prompt`，明确告诉它：“如果一次搜索没有找到，请尝试更换关键词，或根据现有信息回答，不要重复相同的搜索”。

## 生成的结果没有引用来源

agent创建的时候通过设定reponse_format参数可以控制生成内容格式

定义结构化输出实体(pydantic)，作为response_format实参
