短期记忆：当前任务或会话的**上下文**

长期记忆：跨任务会话的**经验和知识**

# AgentState 和 Checkpointer

LangChain 短期记忆通过 AgentState 实现，会话历史，是 AgentState 的一部分

LangChain 提供了 Checkpointer 对象保存 AgentState ，每一次用户与AI的交互生成快照，记录为一个checkpoint

同一个对话的多个 checkpoint 形成一个组，用 thread_id 来标识

## 基于数据库存储 Checkpointer

### 安装依赖

`pip install langgraph-checkpoint-postgres`

### 导入

`from langgraph.checkpoin.postgres import PostgresSaver`

### 初始化 checkpointer

```python
DB_URI = "xxx"
with PostgresSaver.from_conn_string(DB_URI) as checkpointer:
 ...
```

### 自动建表

`checkpointer.setup()`

### 创建智能体，指定 checkpointer

```python
agent = create_agent(
	"deepseek-chat",
	checkpointer=checkpointer,
)
```
