# 方法一：图形界面配置

1. 按 `Win + R`，输入 `sysdm.cpl` 打开系统属性
2. 点击 **高级** 选项卡 → **环境变量**
3. 在用户变量或系统变量中操作：
   - **新建**：添加新变量
   - **编辑**：修改变量值（多个值用 `;` 分隔）
   - **删除**：移除变量

# 方法二：命令行配置（临时生效）

```
set 变量名=变量值        # 当前命令行窗口有效
setx 变量名 变量值       # 永久生效（用户变量）
setx 变量名 变量值 /M    # 永久生效（系统变量，需管理员权限）
```

# 方法三：PowerShell 配置

```powershell
# 设置用户环境变量
[Environment]::SetEnvironmentVariable("变量名", "变量值", "User")

# 设置系统环境变量（需管理员权限）
[Environment]::SetEnvironmentVariable("变量名", "变量值", "Machine")

# 刷新当前会话的环境变量
$env:变量名 = "变量值"
```

# 常用环境变量

| 变量名 | 说明 |
|--------|------|
| `PATH` | 可执行程序搜索路径 |
| `JAVA_HOME` | JDK 安装路径 |
| `MAVEN_HOME` | Maven 安装路径 |
| `PYTHONPATH` | Python 模块搜索路径 |
| `NODE_PATH` | Node.js 模块路径 |
