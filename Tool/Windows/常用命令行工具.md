# 系统信息

| 命令 | 说明 |
|------|------|
| `systeminfo` | 查看系统详细信息 |
| `hostname` | 查看计算机名 |
| `ipconfig /all` | 查看网络配置 |
| `tasklist` | 列出所有进程 |
| `taskkill /IM 进程名 /F` | 强制结束进程 |

# 文件操作

| 命令 | 说明 |
|------|------|
| `dir` | 列出目录内容 |
| `cd` | 切换目录 |
| `mkdir` | 创建目录 |
| `copy` | 复制文件 |
| `xcopy` | 复制目录 |
| `move` | 移动文件/目录 |
| `del` | 删除文件 |
| `rmdir /s /q 目录名` | 强制删除目录及内容 |

# 网络诊断

| 命令 | 说明 |
|------|------|
| `ping 域名/IP` | 测试网络连通性 |
| `tracert 域名/IP` | 追踪路由 |
| `nslookup 域名` | 查询 DNS 解析 |
| `netstat -ano` | 查看网络连接和端口 |
| `telnet IP 端口` | 测试端口连通性 |

# 磁盘管理

| 命令 | 说明 |
|------|------|
| `diskmgmt` | 打开磁盘管理 |
| `chkdsk C: /f` | 检查并修复C盘 |
| `cleanmgr` | 磁盘清理向导 |
