# 操作

按 `Win + R` 键打开运行窗口，输入 `resmon` 回车，打开**资源监视器**。
切换到 **CPU** 选项卡。 
在中间找到 **关联的句柄** 这一栏（可以把它展开）。
在“搜索句柄”的输入框里，输入你这个文件夹的名字
下方立刻就会列出是哪个 `.exe` 程序死死抓着这个文件夹不放。直接在搜索结果里右键那个程序，选择 **结束进程** 即可。

# 安全操作建议

1. 先确认你手头**没有正在进行**的该文件夹相关的“复制、粘贴、渲染导出”任务。
2. 确认相关软件里的工作都已**保存**。
3. 满足这两点后，你就可以放心大胆地去结束进程。即便文件真的因为意外损坏了，大不了重新解压或者重新复制一份即可，**对电脑系统本身和硬件不会有任何伤害。**
