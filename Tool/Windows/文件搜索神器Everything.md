# 简介

**Everything** 是 Windows 上最快的文件搜索工具，可以在秒级时间内搜索全盘文件。

- 官网：https://www.voidtools.com/
- 大小：约 2MB
- 支持：Windows 7 及以上

# 下载安装

1. 访问官网下载安装包
2. 选择安装语言（支持中文）
3. 安装完成后自动建立索引（首次需要几秒到几分钟）

# 基本使用

1. 打开 Everything
2. 在搜索框输入文件名或关键词
3. 结果即时显示

# 搜索语法

| 语法 | 示例 | 说明 |
|------|------|------|
| 空格 | `doc txt` | 同时包含两个词 |
| `\|` | `doc\|txt` | 包含任一词 |
| `!` | `!.mp4` | 排除 |
| `*` | `*.pdf` | 通配符 |
| `"` | `"hello world"` | 精确匹配 |
| `ext:` | `ext:pdf` | 按扩展名搜索 |
| `path:` | `path:D:\project` | 指定路径 |
| `size:` | `size:>100MB` | 按大小过滤 |
| `date:` | `date:today` | 按日期过滤 |

# 实用技巧

## 按文件大小搜索
```
size:>1GB         # 大于 1GB 的文件
size:<100MB       # 小于 100MB 的文件
size:100MB..1GB   # 100MB 到 1GB 之间
```

## 按修改日期搜索
```
dm:today          # 今天修改
dm:yesterday     # 昨天修改
dm:2024/1/1      # 指定日期修改
```

## 组合搜索
```
ext:pdf size:>10MB dm:today   # 今天修改的大于10MB的PDF
```

# 正则表达式

勾选菜单 **搜索 → 启用正则表达式** 后可使用正则：

```
^test.*\.txt$     # 以 test 开头的 txt 文件
\d{4}             # 文件名包含4位数字
```

# 书签功能

可将常用搜索保存为书签：
1. 输入搜索条件
2. 点击 **书签 → 添加到书签**
3. 下次从书签菜单快速调用
