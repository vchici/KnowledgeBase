# 查看资源占用

## 任务管理器
- 按 `Ctrl + Shift + Esc` 打开
- 查看 CPU、内存、磁盘、网络占用
- 按 CPU/内存排序找出高占用进程

## 资源监视器
- 按 `Win + R`，输入 `resmon`
- 更详细的资源监控和进程分析

## Performance Monitor
- 按 `Win + R`，输入 `perfmon`
- 专业性能监控工具

# 常见问题解决

## 磁盘占用 100%

1. 关闭 Windows Search 索引服务：
   - 按 `Win + R`，输入 `services.msc`
   - 找到 Windows Search，设置为 **禁用**

2. 关闭 SysMain（原 SuperFetch）：
   - 服务管理器中找到 SysMain，设为 **禁用**

## 内存不足

1. 查看内存占用进程：`tasklist /FI "MEMUSAGE gt 100000"`
2. 关闭不必要的启动项：
   - 按 `Ctrl + Shift + Esc` → **启动** 选项卡
   - 禁用不需要的开机启动程序

## CPU 占用过高

1. 排查高 CPU 进程
2. 检查是否有异常进程（可能是病毒）
3. 更新或回滚驱动程序

# 系统优化建议

## 清理磁盘空间

```
cleanmgr        # 磁盘清理向导
cleanmgr /sageset:1   # 高级清理选项
```

## 禁用开机启动项

1. 任务管理器 → **启动**
2. 按 `Win + R`，输入 `shell:startup` 查看用户启动文件夹
3. 按 `Win + R`，输入 `msconfig` 管理服务启动

## 清理临时文件

```
%temp%          # 打开临时文件夹，可删除内容
prefetch        # 预读取缓存，可清理
```

## 调整视觉效果提升性能

1. 右键 **此电脑** → **属性** → **高级系统设置**
2. **高级** → **性能** → **设置**
3. 选择 **调整为最佳性能** 或自定义

## 电源计划

1. 按 `Win + R`，输入 `powercfg.cpl`
2. 选择 **高性能** 模式（笔记本会增加耗电）

# 定期维护命令

```
# 检查系统文件完整性（需管理员权限）
sfc /scannow

# 修复系统镜像
DISM /Online /Cleanup-Image /RestoreHealth

# 清理 Windows 更新缓存
Dism.exe /online /Cleanup-Image /StartComponentCleanup
```
