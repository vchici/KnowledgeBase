# 方法一：命令行查找并结束进程

1. 按 `Win + R`，输入 `cmd` 打开命令提示符
2. 查找占用端口的进程：
   ```
   netstat -ano | findstr :端口号
   ```
   例如查看 8080 端口：`netstat -ano | findstr :8080`
3. 找到最后一列的 PID（进程ID）
4. 结束进程：
   ```
   taskkill /PID 进程ID /F
   ```

# 方法二：使用 PowerShell

1. 按 `Win + X`，选择 Windows PowerShell
2. 查找并结束：
   ```powershell
   Get-Process -Id (Get-NetTCPConnection -LocalPort 端口号).OwningProcess | Stop-Process -Force
   ```

# 方法三：使用资源监视器（图形界面）

1. 按 `Win + R`，输入 `resmon`
2. 切换到 **网络** 选项卡
3. 展开 **侦听端口**，找到对应端口
4. 右键对应进程，选择 **结束进程**
