# 通用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Win + E` | 打开文件资源管理器 |
| `Win + D` | 显示/隐藏桌面 |
| `Win + L` | 锁定电脑 |
| `Win + R` | 打开运行对话框 |
| `Win + I` | 打开设置 |
| `Win + A` | 打开操作中心 |
| `Win + S` | 打开搜索 |
| `Win + Tab` | 任务视图 |
| `Win + V` | 剪贴板历史记录 |
| `Win + .` | 打开表情符号面板 |
| `Win + PrintScreen` | 截图并保存到图片文件夹 |
| `Win + Shift + S` | 截图工具 |

# 窗口管理

| 快捷键 | 功能 |
|--------|------|
| `Win + ←/→` | 窗口贴靠到左/右半屏 |
| `Win + ↑/↓` | 最大化/最小化窗口 |
| `Win + Home` | 最小化除当前窗口外的所有窗口 |
| `Win + M` | 最小化所有窗口 |
| `Win + Shift + M` | 还原最小化的窗口 |
| `Alt + Tab` | 切换窗口 |
| `Alt + F4` | 关闭当前窗口 |
| `Ctrl + W` | 关闭当前标签页 |

# 文件资源管理器

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Shift + N` | 新建文件夹 |
| `F2` | 重命名选中文件 |
| `Ctrl + C/X/V` | 复制/剪切/粘贴 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Y` | 重做 |
| `Ctrl + A` | 全选 |
| `Ctrl + Shift + Esc` | 打开任务管理器 |
| `Alt + Enter` | 查看文件属性 |
| `Alt + ←/→` | 后退/前进 |
| `Alt + ↑` | 返回上级目录 |

# 浏览器通用

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + T` | 新建标签页 |
| `Ctrl + W` | 关闭当前标签 |
| `Ctrl + Shift + T` | 恢复关闭的标签 |
| `Ctrl + Tab` | 切换标签 |
| `Ctrl + L` | 聚焦地址栏 |
| `Ctrl + D` | 添加书签 |
| `Ctrl + Shift + Delete` | 清除浏览数据 |
| `F5` | 刷新 |
| `Ctrl + F5` | 强制刷新 |
| `F12` | 开发者工具 |

# 虚拟桌面

| 快捷键 | 功能 |
|--------|------|
| `Win + Ctrl + D` | 新建虚拟桌面 |
| `Win + Ctrl + ←/→` | 切换虚拟桌面 |
| `Win + Ctrl + F4` | 关闭当前虚拟桌面 |

# 文本编辑

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + ←/→` | 按词移动光标 |
| `Shift + ←/→` | 逐字选择 |
| `Ctrl + Shift + ←/→` | 按词选择 |
| `Ctrl + Backspace` | 删除前一个词 |
| `Ctrl + Delete` | 删除后一个词 |
| `Home/End` | 移动到行首/行尾 |
| `Ctrl + Home/End` | 移动到文档开头/结尾 |
