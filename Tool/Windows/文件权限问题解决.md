# 现象

- 删除/移动文件时提示"需要管理员权限"
- 提示"访问被拒绝"
- 无法修改某些系统文件

# 方法一：获取文件所有权

1. 右键文件/文件夹 → **属性**
2. **安全** 选项卡 → **高级**
3. 在"所有者"旁点击 **更改**
4. 输入你的用户名或点击 **高级** → **立即查找** 选择
5. 勾选 **替换子容器和对象的所有者**
6. 点击确定

# 方法二：修改权限

1. 右键文件/文件夹 → **属性**
2. **安全** 选项卡
3. 点击 **编辑**
4. 选择你的用户名
5. 勾选 **完全控制** 的"允许"
6. 点击确定

# 方法三：命令行获取所有权（管理员权限）

```
# 获取文件所有权
takeown /f "文件路径"

# 获取文件夹及其子文件所有权
takeown /f "文件夹路径" /r /d y

# 授予完全控制权限
icacls "文件路径" /grant 用户名:F

# 授予文件夹及其子文件完全控制权限
icacls "文件夹路径" /grant 用户名:F /t
```

# 方法四：解除文件锁定

某些文件被进程占用导致无法操作：

1. 使用 **资源监视器**（`resmon`）查找占用进程
2. 或使用工具 **Unlocker** 强制解除锁定

# 常见场景

## 无法删除文件夹

```
# 强制删除（管理员命令行）
rd /s /q "文件夹路径"
```

## 系统文件无法修改

1. 先获取所有权：`takeown /f "文件路径"`
2. 再授予权限：`icacls "文件路径" /grant 用户名:F`
3. 然后即可修改

## TrustedInstaller 权限问题

某些系统文件所有者是 TrustedInstaller：

1. 按上述方法一，将所有者改为当前用户
2. 修改权限为完全控制
3. 操作完成后建议恢复原权限（可选）
