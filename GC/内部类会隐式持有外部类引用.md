> 非静态内部类必须依附于外部类实例才能创建
> 静态内部类不持有外部类引用

为了非静态内部类能访问外部类成员，内部类必须有一个对外部类的引用，编译器通过在构造函数中注入外部类参数来实现这个机制。

源代码：

```java
class MyActivity {
    private Listener listener = new MyListener();
    
    class MyListener implements Listener {
        void onEvent() {
            updateUI(); // 直接访问外部类方法
        }
    }
}
```

编译后的字节码（反编译）：

```java
class MyActivity {
    private Listener listener = new MyListener(this); // 自动传入 this
    
    class MyListener implements Listener {
        private MyActivity this$0; // 编译器生成的字段
        
        // 编译器生成的构造函数
        MyListener(MyActivity outer) {
            this.this$0 = outer;
        }
        
        void onEvent() {
            this.this$0.updateUI(); // 通过引用调用
        }
    }
}
```
